Some of these are audio instead of AV because the video file wasn't available.


The target file should be comprised of urls to direct lecture capture files, one on each line.
These should be in order from earliest to latest.
A sample target.txt file is included for COMP30023 - Computer Systems.

Make sure to include a cookies.txt from after having logged into the LMS. wget won't be able to access the downloads otherwise.

Also make sure to change the options at the top of the file prior to use.
These relate to things like number of lectures per week, lectures that didn't happen, etc.
